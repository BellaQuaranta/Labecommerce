# LABECOMMERCE

## Essa api tem como funcionalidade a simulação de um e-commerce, onde você pode: 

* Registrar e pegar clientes.
* Adicionar e pegar produtos.
* Registrar e pegar relação de compras de clientes.

## Tecnologias utilizadas:
- MySQL
- Express
- Knex
- Node.js
- Typescript
- Dotenv
- Cors
